# PhotoSift Landing Page Features & Content

Этот документ содержит структурированную информацию для быстрого создания лендинга или сайта (Web), основанную на текущем функционале продукта (Product).

## Главная ценность (Value Proposition)

"Хватит бесконечно листать галерею, чтобы найти свои фото в гигантском экспорте чата на 20 ГБ. PhotoSift использует локальный ИИ, сохраняя вашу приватность, чтобы автоматически найти и отсортировать каждую фотографию с вами."

## Целевая аудитория (Target Audience)

1. **Обычные пользователи**: Люди с огромными архивами из WhatsApp, Telegram или Google Data Takeout, которым просто нужны их собственные фото или фото семьи.
2. **Фотографы и посетители мероприятий**: Участники свадеб, вечеринок или поездок с тысячами кадров, которым нужно быстро раздать фото конкретным гостям.
3. **Ценители приватности (Privacy First)**: Люди, которые не хотят загружать свои личные фото-архивы в облачные сервисы (Google Photos, iCloud) ради распознавания лиц. Всё работает на 100% локально (CPU/GPU).

---

## Ключевые функции для сайта (Features Section)

### 1. 100% Локальная Приватность (Offline AI)

Ваши фотографии никогда не покидают ваш компьютер. Движок распознавания лиц работает полностью офлайн на вашем устройстве, обеспечивая максимальную конфиденциальность и безопасность. Поддержка ускорения на видеокартах (NVIDIA/AMD) для мгновенной сортировки.

### 2. Умное разделение (Solo & Group)

PhotoSift не просто находит вас — он понимает контекст. Программа аккуратно разделяет найденные фотографии:

- **"Только Я" (Solo)**: Идеальные одиночные портреты для аватарок.
- **"Совместные" (With Others)**: Фотографии, где вы находитесь в кампании с друзьями.

### 3. Хронологическая организация (Sort by Year)

Скинули тысячи неразобранных фото из разных источников? Приложение прочитает скрытые метаданные EXIF (дату создания кадра) и автоматически создаст структуру папок по годам (например, `2022`, `2023`), аккуратно разложив все лица по времени.

### 4. Продвинутая фильтрация "Мусора" (Junk & Size Filter)

Мессенджеры часто сохраняют мелкие скриншоты, мемы и элементы интерфейса.

- **Junk Detection**: Автоматически отлавливает файлы меньше 150 КБ, изолируя их в папку "Мусор".
- **Динамический фильтр размера**: Удобный ползунок позволяет задать диапазон веса файла (например, от 0.15 МБ до 5 МБ), отсеивая всё лишнее.

### 5. Поддержка нескольких людей

Нужно разобрать свадебный альбом? Просто добавьте столько людей, сколько нужно — для каждого задайте эталонные фото (Reference Photos). PhotoSift проанализирует весь архив и найдёт фото, на которых присутствуют выбранные люди. Никаких переключателей режимов — добавляйте одного или сотню людей в одном интерфейсе.

### 6. Умные уровни распознавания (Threshold Control)

Настройте алгоритм под ваши нужды с помощью простого ползунка:

- **Мягкий (Soft)**: Найдёт лица в профиль, в темноте или немного размытые.
- **Норма (Normal)**: Идеальный баланс.
- **Строгий (Strict)**: Только 100% уверенные совпадения (без ложных срабатываний).

### 7. Тотальный порядок (Total Clean-up)

PhotoSift не просто ищет вас, он раскладывает ВСЕ файлы "по полочкам". Если на фото нет вашего лица, файл отправится в одну из папок: `Другие люди`, `Без людей` (пейзажи/еда), или `Не распознано` (испорченные файлы). Ни один файл не потеряется.

### 8. Система визуальных пресетов

Сохраняйте любимые конфигурации! Настроили идеальный фильтр для Telegram-архива? Сохраните его как пресет. Приложение помнит все ваши настройки, включая пороги распознавания и эталонные фотографии. Поддержка перевода на 3 языка (EN, RU, UK).
